Configuration Lab

AAA Subnetting
Directions:
Type out the configuration for the router hitting enter for each new line...

Task:

Router 1:
Get into the router, configure hostname R1, on gigabitethernet 0/1 set an ip address for the last possible host for 192.168.1.0/24. Remember that interfaces are shutdown by default, so make sure this one is open. Set a domain name of cisco.com. Generate an rsa key of 2048 bits. Set ssh version to 2. Start AAA. Set the login to admin priv 15 and a secure encrypted password of "cisco". Set the AAA authentication login to default local. Configure 5 vty lines. Set the login authentication to default. Set the protocol to ssh. Do write. Now do show users to check.
PC:
Now, type in the command you would on the pc to connect to the router.

AAA RADIUS TACACS TACACS-GROUP ACL DOT1Q DTP DYNNAT FHRP FTP OSPF0 PAT Portsec QOS SCP SnoopDAI SSH STATNAT SYSLOG VTP
