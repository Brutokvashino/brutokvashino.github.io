ARP Packet:
Preamble (7 bytes): 01010101 x 7. synchronizes receiver's clock
Start Frame Delimiter (SFD) (1 byte): 10101011
Destination MAC (6 bytes), Source MAC (6 bytes)
EtherType (2 bytes): ARP = 0x0806, ipv4 = 0x0800
Payload (ARP Packet): ARP packets typically 28 bytes,
Ethernet min payload 46 bytes, so padding is added if needed.
Frame Check Sequence (FCS) (4 bytes): CRC used for error checking.
