Machine Learning:
Training Data, Test Data = larger set

Ensembles: using multiple algorithms
Bagging: using several versions of the same algorithm
Stacking: using different kinds of algorithms stacked together

Bias: gap between predicted value and actual outcome
Variance: how scattered the predicted values are
Underfitting the data to the model: was not prepared for variables, must make more predictors
Overfitting the data to the model: too many variable

Supervised learning: binary classification of the data requires large and labeled data for the training set to be meaningful.
Unsupervised learning works well for small data set, may identify unexpected connections
The exact methods are unknown.

Artificial Neural Networks: usually supervised learning and self-tuning based on giving each connection in the hidden layer a weight represented by W1 to Wn where n is however many neurons in that layer. Weights are randomly assigned at first but get reassigned after training data fed to tune itself for accuracy. feedforward neural network: input layer>hidden layer>output layer (features neurons with the binary classification each having a probability score). makes guesses based on recognized patterns then compares answer to correct answer expressed as percentage. Useful for breaking down a lot of data.
Cost can be assigned to inform the network on how big an adjustment to make to increase accuracy after errors (measurement of wrongness).
Backpropagation: Neural network adjusts the weights of the connections to lower the cost of the function
The output layer has the probability score
Activation bias: adjusting bias and variance
Overfitting lots of complexity
On the neuron, not the weighted connection. It can only correct after observing outcome
Example:Image recognition used with neural netw, training set, supervised, self tuning weights of connections,

Data Classification: Cluster Data is an output category that can be grouped based on a pattern noted by analyzing the data
KNN = K Nearest Neighbor: plots new data by comparing it with existing data
K Means Clustering: unsupervised and creates clusters in the data over and over to increase accurate cluster based on finding the best centroids in the data. K value = how many clusters desired. Centroids are calculated for distance to all other points.

Reinforcement Learning: using tokens to allow the machine to invent new predictions and solutions.

Euclidean Distance: mathematical formula for distance between data points

Regression Analysis: supervised machine learning algorithm using predictors and outcome. Usually XY diagram plotted to show a trendline at the center of data groupings.
Regressors can be also called independent variables. When to insert product or offer?

Naive Bayes Algorithm: assumes all predictors are independent from one another. Uses predictors, some are autocorrelated but naive considers them all equal. Class predictor probability shows how likely it is to be in such a class. What other product to include?
